### 一、ChatGPT 问答聊天

1. 无需 vpn 直接使用
2. 对当前选中的代码直接进行提问
3. 支持 GPT 角色设定
4. 支持问题的重新生成、删除、拷贝功能
5. 支持一键新建会话
